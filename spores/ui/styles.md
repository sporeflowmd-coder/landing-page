# Styles Spore

## 🎯 Goal
Define the CSS styling that gives the SporeFlow page its fungal/mycelium aesthetic.

## 🎨 Design Direction
- **Theme**: Organic, natural, fungal aesthetic
- **Colors**: Earth tones, mushroom colors (creams, browns, subtle greens)
- **Typography**: Clean, readable, slightly organic feel
- **Visual Elements**: Subtle organic patterns, flowing lines reminiscent of mycelium

## 📥 Input Dependencies
- `spores/ui/layout.md` (Component structure)

## 📤 Output Dependencies
- `/root/src/styles.css` (All CSS rules)
