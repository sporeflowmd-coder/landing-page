# Layout Spore

## 🎯 Goal
Define the HTML structure for the SporeFlow protocol landing page.

## 📐 Layout Components
1. **Header**: Project title and navigation
2. **Hero Section**: Introduction to SporeFlow
3. **Content Sections**: Story chapters (origin, analogy, features, future)
4. **Footer**: Credits and links

## 📥 Input Dependencies
- `spores/content/story.md` (Content to render)
- `spores/manifest.md` (Project metadata)

## 📤 Output Dependencies
- `/root/src/index.html` (HTML structure)
- `/root/src/styles.css` (Layout styles reference)
