# SporeFlow Landing Page

This repository contains the source code for the official landing page of the **SporeFlow Protocol**.

## About SporeFlow

SporeFlow is a protocol designed to guide Large Language Models (LLMs) in software creation using a mycelial analogy. It structures development through "spores" (intent/specs) that manifest into "biomass" (code), providing a framework for:

- **Hallucination Control**: Grounding LLM outputs in structured markdown specifications.
- **Self-Correction**: Learning from errors by "digesting" bugs.
- **Context Management**: Splitting complex logic into manageable units.
- **Secure Isolation**: Keeping secrets and variables in a restricted substrate.

## Project Structure

This project is built as a static HTML site, designed to be lightweight and easily deployable.

## Getting Started

To view the landing page locally, simply open the `index.html` file in your web browser.
